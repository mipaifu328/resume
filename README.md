# 在线预览
 [Demo Live](http://www.mipaifu328.com/resume/dist/)
 
# 安装运行

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run all tests
npm test
```
